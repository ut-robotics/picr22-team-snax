== Mainboard communication

The mainboard receives structs in the form:

typedef struct Command { 
  int16_t speed1;
  int16_t speed2;
  int16_t speed3;
  uint16_t throwerSpeed;
  uint16_t delimiter; 
} Command;

The mainboard returns feedback structs in the form:

Typedef struct Feedback {
  int16_t speed1;
  int16_t speed2;
  int16_t speed3;
  uint16_t delimiter;

} Feedback;
